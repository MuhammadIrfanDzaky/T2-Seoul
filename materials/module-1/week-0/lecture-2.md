### What is a Code Editor?
- A text editor program designed specifically for writing and editing **source code**
- The tool used to implement UI/UX design into an actual result like a website
- Can be a standalone application or part of an IDE

### IDE (Integrated Development Environment)
- A code editor equipped with more complete features beyond a simple text editor
- Source-code editors are a fundamental programming tool

### Popular Code Editors

**CLI Editors** (terminal-based):
- Vi / Vim, Emacs, Nano

**Regular & Lightweight Editors:**
- Visual Studio Code (VS Code) by Microsoft ⭐
- Sublime Text, Notepad++
- Cursor (fork of VS Code, with ✨ vibe coding ✨)

**IDE (Full-featured):**
- Visual Studio — mostly C# and F#
- Android Studio (IntelliJ IDEA) — mostly Java & Kotlin
- Apple Xcode — mostly Objective-C & Swift

### Our IDE: VS Code & Kiro
- **VS Code**: Source code editor by Microsoft, available on Windows, Linux, macOS
- **Kiro**: IDE with key features:
  - Intelligent Code Completion (AI-driven)
  - Built-in Debugging
  - Version Control (Git & GitHub integrated)
  - Advanced Refactoring

### What is Documentation?
- Documentation is a **written explanation about something**; can be text, online, video, or audio
- Contains technical information such as: installation steps, usage guide, specifications, and maintenance
- Analogy: like the user manual that comes in the box when you buy a new phone

### Documentation in Web Development
A web project's documentation typically covers:
- Project name and description
- Who the author / creator / contributor is
- Where the project can be accessed or previewed
- List of pages and features

### README.md
- `README.md` serves as the **main documentation** inside a code repository
- Contains: project name, description, installation steps, and usage guide
- The `.md` extension stands for **Markdown**
- Other common documentation files: `CONTRIBUTING.md`, `HELP.md`

### Rich Text / WYSIWYG
- **WYSIWYG** = *"What You See Is What You Get"* — what you see while editing is the final result
- Called **Rich Text** because it contains heavy formatting (font, color, size, etc.)
- Examples: Microsoft Word, Microsoft PowerPoint, Google Docs, Google Slides

### What is Markdown?
- Markdown is a **lightweight markup language** for adding formatting to plaintext
- Simple — can be written in Notepad or VS Code
- Does not change font, color, or size — focuses on **content**
- Uses keyboard symbols you already know
- Powerful, yet intentionally limited to keep focus on meaningful content writing

### The Power of Markdown
Markdown files can be converted to many output formats:
- PDF, ePub, HTML, LaTeX, CSV, DOC (Word), PPT (PowerPoint)

### How to Use Markdown
- File extension: **`.md`** (most common), `.markdown`, or `.mkd`
- Most Markdown files are named `README.md`
- To **preview** in VS Code: press `Ctrl/Cmd + Shift + P` → type `markdown preview`

### Apa itu Dokumentasi?
- Dokumentasi adalah **penjelasan tertulis tentang sesuatu**; bisa berupa teks, online, video, atau audio
- Berisi informasi teknis seperti: cara instalasi, cara penggunaan, spesifikasi, dan maintenance
- Analogi: seperti buku manual yang ada di dalam kotak ketika kamu beli HP baru

### Dokumentasi dalam Web Development
Dokumentasi proyek web biasanya mencakup:
- Nama dan deskripsi project
- Siapa author / creator / contributor-nya
- Di mana project bisa diakses atau di-preview
- Daftar halaman dan fitur yang tersedia

### README.md
- File `README.md` berfungsi sebagai **dokumentasi utama** di dalam sebuah code repository
- Berisi: nama project, deskripsi, cara instalasi, dan panduan penggunaan
- Ekstensi `.md` merupakan singkatan dari **Markdown**
- File dokumentasi umum lainnya: `CONTRIBUTING.md`, `HELP.md`

### Rich Text / WYSIWYG
- **WYSIWYG** = *"What You See Is What You Get"* — tampilan saat mengedit = tampilan hasil akhir
- Disebut **Rich Text** karena mengandung banyak formatting (font, warna, ukuran, dll)
- Contoh: Microsoft Word, Microsoft PowerPoint, Google Docs, Google Slides

### Apa itu Markdown?
- Markdown adalah **lightweight markup language** untuk menambahkan formatting ke plaintext
- Sederhana — bisa ditulis di Notepad biasa maupun VS Code
- Tidak mengubah font, warna, atau ukuran — fokus pada **konten**
- Menggunakan simbol keyboard yang sudah kamu kenal
- Powerful, namun sengaja dibatasi agar fokus pada penulisan konten yang bermakna

### Kekuatan Markdown
File Markdown bisa dikonversi ke berbagai format output:
- PDF, ePub, HTML, LaTeX, CSV, DOC (Word), PPT (PowerPoint)

### Cara Menggunakan Markdown
- Ekstensi file: **`.md`** (paling umum), `.markdown`, atau `.mkd`
- Sebagian besar file Markdown dinamai `README.md`
- Untuk **preview** di VS Code: tekan `Ctrl/Cmd + Shift + P` → ketik `markdown preview`
